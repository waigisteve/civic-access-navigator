# Demo Script

## Opening

This project is Civic Access Navigator, a capstone aligned with the Open Society Foundations mission. The goal is to help users discover and understand trusted civic information more easily while showing a prompt-aware, workshop-driven development workflow.

## Walkthrough

1. Open the homepage and introduce the project summary.
2. Show the mission focus areas and explain the OSF alignment.
3. Browse the starter civic resource library and point out source metadata.
4. Explain how prompts will be reviewed, adjusted, and checked before release.
5. Show the next implementation step: grounded Q&A over curated resources with source-aware answers.

## Close

The capstone demonstrates a realistic, responsible way to use AI and Codex to improve civic information access without drifting into unverified claims or direct legal advice, and it treats prompting, review, and validation as part of the build process.
