# Capstone Proposal

## Project Title

Civic Access Navigator

## Impact Partner Alignment

This project is aligned with the Open Society Foundations mission to support vibrant and inclusive democracies whose governments are accountable to their people. It focuses on rights and dignity, democratic practice, and equity in governance by improving access to understandable public-interest information.

## Problem Statement

Important civic and public-interest information is often scattered across reports, policy documents, legal aid resources, and community guidance pages. Many potential users do not have the time, technical background, or domain knowledge needed to interpret this information quickly and confidently.

## Goal

Build a lightweight AI-assisted platform that helps users discover, organize, and understand trusted civic resources in plain language.

## Target Users

- students and researchers
- journalists and civic educators
- civil society organizations
- community advocates
- members of the public seeking civic information

## Core Features

- searchable collection of approved civic resources
- resource cards grouped by mission-aligned themes
- plain-language summaries
- project information endpoint for future AI integration
- expandable architecture for retrieval, Q&A, and citations

## Technical Approach

The initial implementation uses:

- FastAPI for the backend
- a lightweight static front end
- structured sample data for civic resources
- an API layer ready for later OpenAI integration

## Why Codex Is a Good Fit

Codex can accelerate:

- repository setup and application scaffolding
- API and UI implementation
- iterative refinement of prompts and workflows
- documentation and evaluation setup
- future integration of retrieval and grounded response patterns

## Scope Control

This capstone intentionally avoids high-risk claims such as legal advice, political prediction, or unverified real-time fact checking. It is framed as an information access tool built on curated, trusted content.

## Expected Impact

The project lowers barriers to civic understanding and participation by making complex information easier to find and interpret. In a practical setting, it can support informed engagement by students, journalists, and community organizations.

## Milestones

1. Set up project structure and working prototype.
2. Curate an initial resource library around one civic theme.
3. Add search, filtering, and plain-language summaries.
4. Integrate grounded Q&A over approved sources.
5. Evaluate output quality and refine the user experience.
