# Project Plan

## Project

Civic Access Navigator

## Partner

Open Society Foundations

## Current Goal

Build a capstone prototype that demonstrates Codex foundations and prompt-aware SDLC practice while helping users discover and understand trusted civic information through a lightweight AI-assisted interface.

## Immediate Priorities

1. Run the starter application locally and confirm the first screen.
2. Replace hardcoded sample resources with file-backed data and source metadata.
3. Add prompt-aware grounded question-answering over curated sources.
4. Add a lightweight review loop for prompt changes, answer checks, and safety checks.
5. Prepare a concise capstone demo flow that shows the build, prompt, and validation steps.
