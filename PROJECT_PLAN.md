# Project Plan

## Project

Civic Access Navigator

## Partner

Open Society Foundations

## Current Goal

Build a PeaceTech capstone prototype that demonstrates Codex foundations and prompt-aware SDLC practice while helping users discover and understand trusted peace and civic information through a lightweight AI-assisted interface.

## Business Goal

Create a reusable PeaceTech product pattern that can adapt to different countries, languages, and local ideas of peace while keeping one trustworthy citation-backed core.

## Immediate Priorities

1. Run the starter application locally and confirm the first screen.
2. Replace hardcoded sample resources with file-backed data and source metadata.
3. Organize resources by PeaceTech track and region, starting with Kenya.
4. Add grounded question-answering over curated sources.
5. Add a lightweight review loop for prompt changes, answer checks, and safety checks.
6. Prepare a concise capstone demo flow that shows the build, prompt, and validation steps.
7. Make the visual system more graphic, cinematic, and attention-grabbing across desktop and mobile.
