# Pitch

## Title

Civic Access Navigator

## Short Pitch

Civic Access Navigator is an OSF-aligned capstone project that helps people find and understand trusted civic information in plain language. It is designed as a practical AI-assisted tool for improving access to rights-focused and governance-related public-interest resources while showing how prompt-aware development can keep the system grounded and reviewable.

## Why It Matters

Important civic information is often difficult to locate, fragmented across sources, or written in language that is hard to interpret quickly. This project lowers that barrier and supports more informed participation.

## What We Are Building

- a lightweight civic information interface
- a curated resource library
- a path toward grounded question-answering over trusted sources
- a prompt-aware build loop with checks for clarity, safety, and source alignment

## Why Codex

Codex helps accelerate scaffolding, iteration, backend logic, frontend changes, and documentation for a capstone that needs to move from concept to working prototype quickly. It also helps us keep prompts, outputs, and verification steps aligned as the project moves through the SDLC.
